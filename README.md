# SerchAdd

Приложение позволяет добавить и сохрнить в UserDefaults адресса, при нажатии на адрес в таблице позволяет проложить до него маршрут.

- UIKit
- MapKit
- CoreLocation
- MVC
- UITableView
- UserDefaults
